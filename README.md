# 学生请假系统 python694

## 项目概述

学生请假系统是一个基于Python+Vue3的前后端分离的Web应用。该系统提供了用户登录、请假管理、学院管理、专业管理、用户管理、公告管理等功能，支持数据的可视化展示和权限的精细控制。

## 技术栈

- 前端：Vue3
- 后端：Python 3.8
- 数据库：MySql 8.x版本
- Node版本：v20.11

## 功能模块

### 用户登录

- 密码找回邮件通知
- 账号页面权限分离

### 用户首页

- 展示用户登录信息
- 系统公告通知

### 数据大屏

- 请假单类型饼图
- 请假次数和时长排行
- 近七日请假申请数

### 管理模块

- 学院管理（增删改查）
- 专业管理（增删改查）
- 用户管理（分为院校领导、辅导员、学生，增删改查，个人信息修改）

### 请假管理

- 学生请假（事假、病假、校外实习、丧假等）
- 超过3天假期需要院校领导审批
- 请假单信息和进度查看

### 待办事宜

- 院校领导和辅导员审批假条
- 审批状态包括通过、拒绝、查看进度

### 角色管理

- 自定义角色名和权限，实现三权分立

### 公告管理

- 配置自定义公告

### 系统日志

- 运行日志
- 安全审计日志

### 系统个性化

- 颜色方案自定义

## 系统角色

- 院校领导
- 辅导员
- 学生

## 运行环境

- Python 3.8

- Node v20.11
